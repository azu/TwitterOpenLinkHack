# TwitterOpenLinkHack

* Twitter-for-Mac SIMBL plugin

## Feature

* Open links in **foreground**

## Installation

* Install [EasySIMBL](https://github.com/norio-nomura/EasySIMBL/ "EasySIMBL")
* Drog & Drop ``TwitterOpenLinkHack.bundle`` to SIMBL folder
